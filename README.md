Назначение

Скрипт предназначен для анализа данных опроса студентов, посвященного использованию ИИ-сервисов (ChatGPT, Google Gemini, DeepSeek, YandexGPT / Алиса, Microsoft Copilot). Он выполняет следующие задачи:

Рассчитывает интегральные оценки ИИ-сервисов для каждого студента по шести критериям: удобство использования, качество ответов, полезность для учёбы, полезность для программирования, скорость ответа и доверие к ответам.
Создает характеристики экспертов (студентов), включая цифровую компетентность, частоту использования ИИ, количество сервисов, число учебных задач, ускорение выполнения заданий, понимание сложных тем, доверие к ИИ и желание чаще использовать ИИ.
Проводит факторный анализ для оценок ИИ-сервисов и характеристик студентов:
Тест сферичности Бартлетта для проверки корреляций.
Мера KMO для оценки пригодности данных.
PCA и вращение Varimax для выявления скрытых факторов.
Проверяет согласованность оценок студентов:
Критерий Фридмана для статистической проверки различий между сервисами.
Коэффициент конкордации Кенделла (W) до и после фильтрации выбивающихся респондентов.
Строит регрессионную модель для предсказания средней оценки ИИ-сервисов на основе характеристик студентов:
Метод наименьших квадратов (LinearRegression).
Вычисляет коэффициенты модели и R
