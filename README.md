### Назначение
Скрипт предназначен для анализа данных опроса студентов о использовании ИИ-сервисов (ChatGPT, Google Gemini, DeepSeek, YandexGPT / Алиса, Microsoft Copilot). Он выполняет:

1. Расчет интегральных оценок ИИ-сервисов для каждого студента по шести критериям:
   - Удобство использования
   - Качество ответов
   - Полезность для учебы
   - Полезность для программирования
   - Скорость ответа
   - Доверие к ответам сервиса

2. Создание характеристик студентов:
   - Цифровая компетентность
   - Частота использования ИИ
   - Количество используемых сервисов
   - Количество учебных задач
   - Ускорение выполнения заданий
   - Понимание сложных тем
   - Доверие к ИИ
   - Желание чаще использовать ИИ

3. Проведение факторного анализа:
   - Тест сферичности Бартлетта
   - Мера KMO
   - PCA и Varimax для выявления скрытых факторов

4. Проверка согласованности оценок студентов:
   - Критерий Фридмана
   - Коэффициент конкордации Кенделла (W)
   - Фильтрация выбивающихся респондентов

5. Построение регрессионной модели для предсказания средней оценки ИИ-сервисов на основе характеристик студентов:
   - Линейная регрессия
   - Вычисление коэффициентов модели и R²

### Используемые библиотеки
- `pandas`, `numpy`, `scipy`, `sklearn`, `json`, `collections`, `pathlib`
